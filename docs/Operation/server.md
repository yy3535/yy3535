## http-server
- 启动一个服务查看静态页面
```js
npm i http-server -g
cd到需要查看的页面的文件夹处（该文件夹必须是包含js、css等所有文件的文件夹）
http-server -h(可以看一下server下的命令都有什么)
http-server . -c-1（-c-1是不需要缓存）       
```
