# linux

基本指令

ssh root@xxx 远程连接

ls -a 查看所有文件 包括隐藏

mkdir xx 创建目录

rmdir xx 删除目录

rm -rf xx 强制删除文件

cd 切换

pwd 显示目录

cp [源文件或者目录] [目标文件]

mv [源文件或者目录] [目标文件]

ln -s [源文件] [目标文件] -s创建软连接 类似Windows快捷方式 软连接权限都是777 修改任意一个文件，另一个都会改变 软链接源文件必须写绝对路径

find find [搜索范围] [搜索条件] 是在系统当中搜索符合条件的文件名，如果需要匹配，使用通配符匹配，通配符是完全匹配

vim xx 编辑文件

cat 查看文件
